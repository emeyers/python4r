# Python for R Users
## JSM 2026 Continuing Education Workshop

**Date:** Monday, August 3, 2026 | 1:00–5:00 PM  
**Instructor:** Ethan Meyers  
**Website:** https://emeyers.github.io/python4r/

This workshop introduces Python to statisticians and data scientists who already know R. We focus on the differences that matter, the pitfalls that catch people moving from R, and the central Python data science libraries (NumPy, Pandas, Matplotlib, Seaborn, StatsModels, scikit-learn).

---


## Before the Workshop

Please do the following prior to the start of the workshop: 

1. Complete a background survey to help tailor the workshop to everyone's interests (the sooner you can do this the better). A link to the survey will be added soon.
2. Get the workshop material onto your computer. No terminal required:
   1. [Download everything as a ZIP file](https://github.com/emeyers/python4r/archive/refs/heads/main.zip)
   2. Unzip it. On Windows, right-click the file and choose **Extract All**. On a Mac, double-click it.
   3. Open the new `python4r-main` folder and double-click **`START-HERE-Windows.bat`** (Windows) or **`START-HERE-Mac.command`** (Mac). The first run installs Python and the course packages, which takes about 5 to 10 minutes, and then starts JupyterLab in your browser.

   The first time you run the launcher your computer may ask permission. On a Mac, control-click the file and choose **Open**. On Windows, click **More info** and then **Run anyway**. See the [workshop website](https://emeyers.github.io/python4r/#setup) for full details, or use [Google Colab](https://emeyers.github.io/python4r/#materials) if you would rather not install anything.
